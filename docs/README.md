## App Overview

**BabyName** is a cross-platform (iOS, Android, Web) application that helps expectant parents discover, evaluate, and save baby name options. It is powered by 100+ years of U.S. Social Security Administration (SSA) birth name data, providing genuine popularity trend information for every name.

### Core Screens

1. **Browse** — A scrollable, filterable, alphabetical list of all names with click-through trend detail views.
2. **Swipe** — A swipe-card interface (right to keep, left to pass) that cycles through names randomly and saves decisions.
3. **Account & Lists** — User authentication with the ability to save, organize, and review name lists.
